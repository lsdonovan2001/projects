# College Basketball Performance Modeling (2015)

**Tools:** R, Regression Analysis, Data Wrangling, Feature Engineering

This project analyzes NCAA Division I basketball data to determine which team metrics are most predictive of winning percentage during the 2015 season.

### Key Components
- Cleaned and prepared NCAA team statistics dataset.
- Selected predictors including offensive/defensive efficiency, turnover rate, rebound percentage, and shooting rates.
- Built and evaluated regression models to quantify variable importance.
- Identified performance metrics most strongly associated with success.
